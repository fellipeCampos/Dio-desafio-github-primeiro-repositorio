Caminho 

*Criar um repositório no github
*Abrir o gtibash (de preferencia na pasta onde será colocado o projeto)
*comando = git clone "link do projeto copiado do github" ( sem aspas)
*git status = mostra se o projeto precisa ser commitado ou push etc..
* git add . ou -A = Adiciona tudo que estiver na pasta para ser commitado
* git commit -m "Mensagem coerente com a alteração" = commita o projeto alterado ja deixando ele pronto para subir para o repositorio remoto
* git push origin main (ou master)
